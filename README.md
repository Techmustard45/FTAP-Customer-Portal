This is my final project for Truecoders coding bootcamp.

This project is a web app that shows off race times in an easy to read fashion. It includes a full leaderboard and ranking of all times in the database,
A searchbar to look at specific racers,
An option to update customer info,
and ability to add laptimes to the database.

I plan to expand on this project further to include:
comparing racers,
auto-updating using a tracks API,
and more.
